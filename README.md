# proxima-hub · `app.proxima.pe`

El **Hub web** de Próxima: identidad centralizada (login/registro/recuperar/verificar), launcher de productos (app-switcher) y cuenta/billing de la suite.

- **Arquitectura:** [ARCHITECTURE.md](./ARCHITECTURE.md)
- **Spec + tareas:** [openspec/changes/platform-app-shell](./openspec/changes/platform-app-shell)
- **Stack:** Angular 21 · PrimeNG 21 · Tailwind v4 · Fraunces/Plus Jakarta/Inter
- **Diseño:** claro, plano, editorial, premium. Sin gradientes/glow/glass.
- **Backend:** consume `proxima-api` (`/auth`, `/me`, `/billing`, business_status). No reescribe backend.
